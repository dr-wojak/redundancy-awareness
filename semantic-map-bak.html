Interpretive synthesis · three layers, one directed labelled graph
The ontology encodes the causal chain. The failure modes instance it. The dissemination channels route it. Three pages — one argument.
Deontological Differences establishes the dissemination architecture: five channels, each with its own obligation and legal grounding, each routing the same master causal chain to a distinct audience. Ontology Relations formalises that chain as OWL 2 object properties — directional causal operators with explicit axioms. Failure Modes instantiates each relation with a concrete, repeatable institutional output. The three pages are not parallel documents but layers of a single directed labelled graph: class hierarchy above, causal operators as edges, failure instances as nodes below, and five dissemination channels as terminal outputs. When the SPARQL query "find all violations caused by ignored submissions" runs against this corpus, it returns a structured list directly admissible in CJEU Member State failure proceedings. Nineteen years of narrative argument — now machine-readable institutional critique.
